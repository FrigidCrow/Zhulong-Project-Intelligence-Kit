# Phase Record Schema

Recommended phase files:

- `<phase>-CONTEXT.md`
- `<phase>-RESEARCH.md`
- `<phase>-SPEC.md`
- `<phase>-PLAN.md`
- `<phase>-VALIDATION.md`
- `<phase>-SUMMARY.md`

Minimum phase content:

- Objective
- Scope
- Out of scope
- Relevant context
- Graph/RAG evidence
- Source verification
- Plan
- Execution notes
- Verification
- AI-PIKit Evidence Writeback entries when evidence was recorded separately
- Risks
- Follow-ups
