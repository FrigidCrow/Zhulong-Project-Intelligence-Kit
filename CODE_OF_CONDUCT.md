# 社区行为规范

Zhulong 社区欢迎不同经验、背景和观点的参与者。所有参与者都应共同维护专业、友善和可协作的环境。

## 期望行为

- 尊重他人，围绕事实、代码和可复现证据讨论问题。
- 对反馈保持开放，并清楚区分个人偏好与项目约束。
- 保护他人的隐私、凭据和未公开资料。
- 在发现错误时帮助修正，而不是攻击提出问题的人。
- 对自己的言行及其影响负责。

## 不可接受的行为

- 骚扰、歧视、威胁、侮辱或持续的人身攻击。
- 未经允许公开他人的身份信息、密钥、私有代码或客户资料。
- 故意破坏讨论、Issue、Pull Request、发布或安全响应。
- 在被明确要求停止后继续进行不受欢迎的接触。

## 执行

维护者可以编辑、隐藏或移除违反本规范的内容，并根据严重程度限制参与权限。涉及安全问题时，请按照 [SECURITY.md](SECURITY.md) 私密报告；其他行为问题可以通过仓库维护者的 GitHub 主页联系。

本规范适用于仓库、Issue、Pull Request、Discussion 以及代表 Zhulong 参与的其他公开空间。
