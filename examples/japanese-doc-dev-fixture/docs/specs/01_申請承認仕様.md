# 申請承認仕様

文書ID: SPEC-APP-001  
対象: 購買申請承認サービス  
最終更新: 2026-06-18

## 1. 用語

| 用語 | 説明 | コード上の別名 |
| --- | --- | --- |
| 申請 | 購買、経費、または社内依頼の承認対象 | request |
| 承認 | 申請を処理可能状態に進める操作 | approval |
| 代理承認 | 本来の承認者に代わって承認する操作 | proxy approval |
| 上長承認 | 30,000円を超える申請に必要な追加承認 | manager approval |

## 2. 承認金額ルール

- REQ-APP-010: 30,000円以下の通常申請は自動承認できる。
- REQ-APP-020: 30,000円を超える通常申請は上長承認を必要とする。
- REQ-APP-030: 申請金額が0円以下の場合は却下する。
- REQ-APP-040: 代理承認者による承認は、QA-042およびCR-017により30,000円以下に制限する。

## 3. 補足

旧版仕様では代理承認の上限が50,000円と読める記述が残っていた。
2026-06-18の議事録で、代理承認の上限金額は30,000円へ統一することが決定された。
実装変更時はQA票と議事録を根拠として扱うこと。
