# 2026-06-18 承認仕様定例 議事録

参加者: PO, 業務担当, 開発リード, QA担当  
対象リリース: 2026年7月保守リリース

## 決定事項

1. CR-017として代理承認上限の改修を実施する。
2. 代理承認の上限金額は30,000円へ統一する。
3. 旧仕様にある50,000円の記述は廃止扱いとし、実装根拠にしない。
4. 影響範囲は承認判定ロジック、監査メモ、承認系テストに限定する想定。

## 宿題

- 開発チームはコードマップで `evaluateApproval` の呼び出し元を確認する。
- QA担当は30,000円、30,001円、45,000円の境界値をテスト仕様に追加する。
- 実装後はテスト結果と参照したQA票を証跡として残す。
