# QA-042 代理承認上限

起票日: 2026-06-12  
回答日: 2026-06-18  
関連: CR-017, REQ-APP-040

## 質問

基本設計書の一部に「代理承認は50,000円まで可能」と読める記述があります。
一方、申請承認仕様では通常承認の自動承認上限が30,000円です。
代理承認の上限金額は50,000円と30,000円のどちらを採用しますか。

## 回答

代理承認の上限金額は30,000円とする。
50,000円の記述は旧運用からの残存であり、今回のリリースでは採用しない。
30,000円を超える申請は、代理承認者による承認ではなく、正式承認者または上長承認ルートへ回す。

## 実装メモ

- `PROXY_APPROVAL_LIMIT` は30,000へ変更する。
- 30,001円以上の代理承認は `PROXY_LIMIT_EXCEEDED` として却下する。
- 上長承認済みであっても、代理承認者が処理する場合は代理承認上限を優先する。
