# CR-017 代理承認上限の変更

## 背景

QA-042および2026-06-18議事録により、代理承認の上限金額は30,000円へ統一された。
現行コードは旧仕様の50,000円を保持しているため、30,001円以上の代理承認を許可する可能性がある。

## 変更内容

- `src/approvalPolicy.js` の `PROXY_APPROVAL_LIMIT` を30,000へ変更する。
- 30,000円を超える代理承認は `PROXY_LIMIT_EXCEEDED` として却下する。
- 上長承認済みであっても代理承認上限を優先する。
